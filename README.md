
## 一个简易的 react 的项目练手(增删改查)

react + react-router + react-redux + node + mongodb

### 安装
```bash
$ git clone git@github.com:bowmansir/react-curd.git
$ cd react-curd
$ npm install # 或者 yarn install
$ npm start
$ cd backend
$ npm install
$ npm start
```
注意:  数据库用的是 `mongodb`,  默认地址是: `127.0.0.1`, 自行在 `serve.js` 中更改
